This project is built using a combination of modern frontend technologies and powerful libraries:

*   **Core:**
    *   **HTML5:** Semantic structure and content.
    *   **CSS3:** Advanced styling with custom properties (CSS variables), Flexbox, CSS Grid, and complex keyframe animations. Uses a custom "Neon Glassmorphism" design system.

    *   **Three.js (v0.132.2):** For rendering interactive 3D graphics and models.

    *   **GLTFLoader:** For loading 3D models.
    *   **Chart.js:** For displaying data visualizations and stats.
    *   **Particles.js (v2.0.0):** To create dynamic particle background effects.
    *   **Typed.js (v2.0.12):** For engaging typewriter text effects.
    *   **Lottie Player:** For lightweight, scalable vector animations.
    *   **Animate.css (v4.1.1):** For pre-built, cross-browser animations.
    *   **FontAwesome (v6.4.0):** For a comprehensive library of vector icons.
